# Location Map Image

## 지도 이미지 추가 방법

1. 네이버 지도에서 매장 위치 검색
   - 주소: 경남 창원시 의창구 의창대로261번길 6 1층

2. 지도 화면 스크린샷 촬영

3. 스크린샷 파일을 이 폴더에 `map.jpg` 이름으로 저장

## 권장 이미지 사양

- 파일명: `map.jpg` (또는 `map.png`)
- 권장 해상도: 최소 1200px × 800px
- 파일 형식: JPG 또는 PNG
- 파일 크기: 500KB 이하 권장

## 현재 상태

⚠️ **지도 이미지가 아직 추가되지 않았습니다.**

웹사이트를 정상적으로 표시하려면 이 폴더에 `map.jpg` 파일을 추가해주세요.
