# Hero 배경 이미지

## 이미지 추가 방법

이 폴더에 `hero-bg.jpg` 파일을 추가하세요.

## 권장 이미지 사양

- **파일명**: `hero-bg.jpg` (필수)
- **해상도**: 1920px × 1080px 이상 (Full HD)
- **비율**: 16:9 권장
- **파일 크기**: 500KB ~ 1MB (웹 최적화 권장)
- **형식**: JPG 또는 PNG

## 이미지 내용 추천

- 숙성 삼겹살 구이 장면
- 매장 인테리어 전경
- 고기 부위별 컷
- 음식 플레이팅 샷

## 현재 상태

⚠️ **Hero 배경 이미지가 아직 추가되지 않았습니다.**

이 폴더에 `hero-bg.jpg` 파일을 추가하시면 메인 화면 배경으로 자동 적용됩니다.
